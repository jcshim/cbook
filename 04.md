16. 게임 기획: 탁구는 어떤 게임일까?

이제 우리만의 게임을 만들어 볼 거예요!  
첫 번째 프로젝트는 바로 **탁구 게임 (Pong)** 입니다.

탁구 게임은 다음과 같은 요소로 이루어져 있어요:
- 왼쪽과 오른쪽에 있는 **패들(Paddle)**  
- 이리저리 튀어다니는 **공(Ball)**  
- 공이 튕기면서 **점수**가 올라가는 시스템

🎯 우리가 만들어갈 단계
1. 공을 그린다
2. 공이 움직인다
3. 벽과 튕긴다
4. 패들을 움직인다
5. 공과 패들이 부딪힌다
6. 점수와 게임 끝을 만든다

이제 1단계부터 함께 만들어 볼까요?


17. 공과 패들 그리기

먼저 공 하나, 패들 하나를 화면에 그려 볼 거예요.

🔍 예시:

```c
#include "raylib.h"

int main() {
    InitWindow(800, 600, "탁구 게임 시작!");
    SetTargetFPS(60);

    int ballX = 400, ballY = 300;
    int ballSize = 10;

    int paddleX = 20, paddleY = 250;
    int paddleWidth = 20, paddleHeight = 100;

    while (!WindowShouldClose()) {
        BeginDrawing();
        ClearBackground(BLACK);

        // 공 그리기
        DrawCircle(ballX, ballY, ballSize, WHITE);

        // 패들 그리기
        DrawRectangle(paddleX, paddleY, paddleWidth, paddleHeight, GREEN);

        EndDrawing();
    }

    CloseWindow();
    return 0;
}
```

🎯 미션!
- 공과 패들의 색을 바꿔보기
- 패들 위치를 바꿔보기


18. 공 튀기기: 벽에 부딪히기

이제 공을 **움직이게** 해 보고, 벽에 닿으면 **튕기게** 만들어 볼 거예요.

🔍 예시:

```c
#include "raylib.h"

int main() {
    InitWindow(800, 600, "공 튕기기");
    SetTargetFPS(60);

    int ballX = 400, ballY = 300;
    int ballSpeedX = 4, ballSpeedY = 4;
    int ballSize = 10;

    while (!WindowShouldClose()) {
        ballX += ballSpeedX;
        ballY += ballSpeedY;

        // 위 아래 벽에 닿으면 방향 바꾸기
        if (ballY <= 0 || ballY >= 600) {
            ballSpeedY *= -1;
        }

        BeginDrawing();
        ClearBackground(BLACK);
        DrawCircle(ballX, ballY, ballSize, WHITE);
        EndDrawing();
    }

    CloseWindow();
    return 0;
}
```

🎯 미션!
- 공의 속도를 바꿔보기
- 좌우 벽에 닿으면 게임 종료 메세지 출력해 보기


19. 키보드로 패들 움직이기

패들을 위아래로 움직일 수 있어야 게임이 되겠죠?

🔍 예시:

```c
#include "raylib.h"

int main() {
    InitWindow(800, 600, "패들 움직이기");
    SetTargetFPS(60);

    int paddleY = 250;
    int paddleSpeed = 5;

    while (!WindowShouldClose()) {
        if (IsKeyDown(KEY_UP)) paddleY -= paddleSpeed;
        if (IsKeyDown(KEY_DOWN)) paddleY += paddleSpeed;

        BeginDrawing();
        ClearBackground(BLACK);

        DrawRectangle(20, paddleY, 20, 100, GREEN);

        EndDrawing();
    }

    CloseWindow();
    return 0;
}
```

🎯 미션!
- 패들이 화면 밖으로 못 나가게 만들기
- 움직이는 속도 조절하기


20. 공과 패들 충돌 감지하기

공이 패들에 닿으면 **튕기게** 만들어야 해요.  
raylib에서는 사각형과 원의 위치를 비교해서 충돌을 직접 계산해야 해요.

🔍 예시:

```c
#include "raylib.h"

int main() {
    InitWindow(800, 600, "충돌 감지");
    SetTargetFPS(60);

    int ballX = 400, ballY = 300;
    int ballSpeedX = -4, ballSpeedY = 4;
    int ballSize = 10;

    int paddleX = 20, paddleY = 250;
    int paddleWidth = 20, paddleHeight = 100;

    while (!WindowShouldClose()) {
        ballX += ballSpeedX;
        ballY += ballSpeedY;

        // 벽 반사
        if (ballY <= 0 || ballY >= 600) ballSpeedY *= -1;

        // 키 입력
        if (IsKeyDown(KEY_UP)) paddleY -= 5;
        if (IsKeyDown(KEY_DOWN)) paddleY += 5;

        // 충돌 체크
        if (ballX - ballSize <= paddleX + paddleWidth &&
            ballY >= paddleY &&
            ballY <= paddleY + paddleHeight) {
            ballSpeedX *= -1;
        }

        BeginDrawing();
        ClearBackground(BLACK);

        DrawCircle(ballX, ballY, ballSize, WHITE);
        DrawRectangle(paddleX, paddleY, paddleWidth, paddleHeight, GREEN);

        EndDrawing();
    }

    CloseWindow();
    return 0;
}
```

🎯 미션!
- 충돌 시 공 속도 바꾸기
- 공의 방향을 랜덤하게 바꿔보기


21. 점수 계산과 게임 종료 만들기

마지막으로 **점수**를 표시하고, **게임 오버** 메시지도 만들어 볼 거예요.

🔍 예시:

```c
#include "raylib.h"

int main() {
    InitWindow(800, 600, "점수와 게임 오버");
    SetTargetFPS(60);

    int ballX = 400, ballY = 300;
    int ballSpeedX = -4, ballSpeedY = 4;
    int ballSize = 10;

    int paddleY = 250;

    int score = 0;

    while (!WindowShouldClose()) {
        // 키 입력
        if (IsKeyDown(KEY_UP)) paddleY -= 5;
        if (IsKeyDown(KEY_DOWN)) paddleY += 5;

        // 공 이동
        ballX += ballSpeedX;
        ballY += ballSpeedY;

        // 벽 반사
        if (ballY <= 0 || ballY >= 600) ballSpeedY *= -1;

        // 충돌 감지
        if (ballX - ballSize <= 40 && ballY >= paddleY && ballY <= paddleY + 100) {
            ballSpeedX *= -1;
            score += 1;
        }

        BeginDrawing();
        ClearBackground(BLACK);

        DrawCircle(ballX, ballY, ballSize, WHITE);
        DrawRectangle(20, paddleY, 20, 100, GREEN);

        DrawText(TextFormat("점수: %d", score), 650, 20, 20, YELLOW);

        if (ballX < 0) {
            DrawText("게임 오버!", 300, 250, 40, RED);
        }

        EndDrawing();
    }

    CloseWindow();
    return 0;
}
```

🎯 미션!
- 10점이 되면 "승리!" 메시지 출력하기
- 다른 쪽에 컴퓨터 패들 만들기 (도전!)
